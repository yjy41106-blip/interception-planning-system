# 前端应用

纯 HTML + Canvas 实现的实时可视化系统。

## 快速开始

### 方式 1：直接打开（推荐）

```bash
# Windows
双击 index.html

# macOS / Linux
open index.html
```

### 方式 2：用浏览器打开

```bash
# 任何浏览器
firefox index.html
# 或
chrome index.html
```

## 功能说明

- **实时可视化**：Canvas 2D 绘制
- **参数调整**：左侧滑块控制
- **Mock 数据**：无需后端即可演示
- **后端联调**：自动检测后端服务

## 无需配置

- 无需 npm install
- 无需构建
- 无需服务器
- 双击即用

## 后端集成

如果启动了后端服务（http://localhost:8000），前端会自动调用真实 API。

否则，前端使用内置 Mock 数据演示。
